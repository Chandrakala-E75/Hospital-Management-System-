Project Overview
This project aims to provide a comprehensive hospital management system through a website. The system facilitates appointment scheduling for patients and includes features such as a home page, about section, contact form, and an admin panel.
Key Features
Appointment Scheduling: Patients can easily schedule appointments with healthcare providers.
User Roles:
Admin Panel: Accessible to hospital receptionists for managing appointments and queries.
Patient/User Interface: Allows patients and users to access information and submit queries through the contact page.
Contact Form:
Users can submit queries or requests through the contact form.
Admins review and respond to queries via email or phone.
echnologies Used
